# Dampol - Premium Container Architecture Website

This project is built with **Astro**, **Tailwind CSS**, and **React** (for animations).

## Tech Stack

- **Framework**: Astro 5 (Zero JS by default, islands architecture)
- **Styling**: Tailwind CSS v4
- **Interactivity/Animations**: React + Framer Motion
- **Icons**: Lucide React
- **Fonts**: Manrope (Google Fonts)

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Project Structure

- `src/pages`: Astro pages (Routing)
- `src/components`: UI Components (Astro & React)
- `src/layouts`: Main Layout (SEO, Meta tags)
- `src/styles`: Global styles (Tailwind theme)
- `public/assets`: Images and static assets

## SEO Features

- Semantic HTML5 structure
